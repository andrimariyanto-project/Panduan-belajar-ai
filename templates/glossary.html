Kamus AI

Istilah yang sering muncul di dunia AI, dijelaskan singkat tanpa basa-basi.

LLM
Large Language Model — model AI yang dilatih pada teks dalam jumlah besar untuk memprediksi dan menghasilkan bahasa.
Token
Satuan terkecil teks yang diproses model, bisa berupa kata utuh, pecahan kata, atau tanda baca.
Context Window
Jumlah maksimum token yang bisa "diingat" model dalam satu percakapan atau permintaan.
Prompt Engineering
Praktik menyusun instruksi ke AI agar menghasilkan output yang akurat dan sesuai kebutuhan.
Hallucination
Kondisi saat AI menghasilkan informasi yang terdengar meyakinkan tapi salah atau tidak berdasar fakta.
Fine-tuning
Proses melatih ulang model dasar dengan data spesifik agar lebih akurat pada tugas tertentu.
RAG
Retrieval-Augmented Generation — teknik menggabungkan pencarian dokumen dengan generasi teks agar jawaban AI berdasar sumber yang relevan.
Embedding
Representasi numerik dari teks/gambar yang menangkap makna, digunakan untuk pencarian kemiripan.
Vector Database
Database yang dioptimalkan untuk menyimpan dan mencari embedding berdasarkan kemiripan makna, bukan kecocokan kata persis.
AI Agent
Sistem AI yang bisa merencanakan langkah, memanggil tools/API, dan bertindak menuju tujuan tanpa instruksi manual di setiap langkah.
Function Calling / Tool Use
Kemampuan model memanggil fungsi atau API eksternal untuk mengambil data atau menjalankan aksi nyata.
Few-shot / Zero-shot
Cara memberi contoh (few-shot) atau tanpa contoh sama sekali (zero-shot) dalam prompt untuk memandu output model.
Temperature
Parameter yang mengatur seberapa acak/kreatif output model — makin tinggi, makin variatif jawabannya.
Inference
Proses model menghasilkan output dari input yang diberikan, setelah tahap pelatihan selesai.
Prompt Chaining
Menyusun beberapa prompt secara berurutan, di mana output satu langkah jadi input langkah berikutnya.
MCP (Model Context Protocol)
Protokol standar yang memungkinkan AI terhubung dan menggunakan tools/data eksternal secara konsisten.
Open-weight Model
Model AI yang bobot (parameter) nya dipublikasikan sehingga bisa dijalankan atau dimodifikasi sendiri, berbeda dari model tertutup.
Guardrail
Batasan atau aturan yang dipasang di sekitar sistem AI untuk mencegah output yang tidak diinginkan atau berbahaya.
Latency
Waktu yang dibutuhkan sistem AI untuk merespons — penting untuk aplikasi real-time seperti monitoring jaringan.
Bias (AI)
Kecenderungan model menghasilkan output yang condong ke pola tertentu akibat data latih yang tidak seimbang.
Synthetic Data
Data buatan (bukan dari kejadian nyata) yang dihasilkan untuk melatih atau menguji model.
Eval (Evaluation)
Proses sistematis mengukur kualitas output AI terhadap kriteria atau dataset uji tertentu.
Context Engineering
Praktik menyusun informasi (dokumen, riwayat, data) yang diberikan ke model agar hasilnya lebih relevan dan akurat.
Multimodal
Kemampuan model memproses lebih dari satu jenis data sekaligus, misalnya teks, gambar, dan audio.
Chain-of-Thought (CoT)
Teknik prompting yang mendorong model menuliskan langkah penalaran secara bertahap sebelum sampai ke jawaban akhir.
ReAct
Pola agent yang bergantian antara bernalar (reasoning) dan mengambil aksi/tool (acting) hingga tujuan tercapai.
Self-Consistency
Teknik mengambil beberapa jawaban dari model untuk prompt yang sama, lalu memilih jawaban yang paling konsisten muncul.
Prompt Injection
Serangan yang menyisipkan instruksi tersembunyi ke dalam input/dokumen untuk membajak perilaku model AI.
Reranking
Tahap menyaring ulang hasil pencarian/retrieval agar dokumen paling relevan naik ke urutan atas sebelum diproses model.
Chunking
Proses memecah dokumen panjang menjadi potongan-potongan kecil sebelum diubah jadi embedding dalam sistem RAG.
Hybrid Search
Pencarian yang menggabungkan pencocokan kata kunci (keyword) dengan pencarian makna (semantic/vector) untuk hasil lebih akurat.
LLM-as-Judge
Pendekatan evaluasi di mana satu model AI dipakai untuk menilai kualitas output model AI lain berdasarkan rubrik tertentu.
Reflection (Agentic Pattern)
Pola desain agent di mana AI mengevaluasi dan mengkritik hasil kerjanya sendiri sebelum memberikan jawaban final.
Groundedness
Ukuran seberapa kuat jawaban AI didukung oleh sumber/dokumen nyata, bukan hasil karangan model semata.
