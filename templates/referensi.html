Referensi & Jalan Pintas Belajar

Kurasi sumber belajar AI eksternal yang resmi dan aktif diperbarui — dipakai sebagai pelengkap roadmap, bukan pengganti. Semua tautan mengarah ke dokumentasi resmi vendor atau course gratis yang sudah dicek ketersediaannya. Kalau kamu cuma punya waktu terbatas, ini "jalan pintas" paling efisien untuk paham AI secara mendalam tanpa tersesat di lautan konten.

> jalur_kilat.plan

Jalur Kilat 30-60-90 Hari

Hari 1–30 · Fondasi

Selesaikan node_00 di Roadmap, lalu tuntaskan AI Prompting for Everyone (DeepLearning.AI) dan baca Panduan Prompt Engineering Anthropic. Latih dengan 10 prompt kategori 🎓 Pembelajaran sub-topik "Fondasi AI" di Prompt Library.

Hari 31–60 · Spesialisasi & Sistem

Ambil ChatGPT Prompt Engineering for Developers, lalu satu course RAG (LangChain/LlamaIndex). Praktikkan lewat 10 prompt sub-topik "RAG" dan "AI Agent & MCP", plus workflow di Workflows.

Hari 61–90 · Produksi & Portofolio

Baca dokumentasi resmi Model Context Protocol, bangun 1 project dari Projects dengan bimbingan sub-topik "Project Praktik", lalu susun studi kasus di Monetize.

Dokumentasi & Panduan Resmi Vendor

Sumber paling akurat karena ditulis langsung oleh tim yang membangun modelnya — dipakai sebagai rujukan utama tiap kali istilah di Kamus AI atau prompt di sini terasa kurang jelas.

resmiAnthropic

Prompt Engineering Overview (Claude Docs)

Dokumentasi teknik inti: instruksi jelas, multishot example, chain-of-thought, XML tag, hingga strategi context panjang — sumber utama gaya penulisan prompt di Prompt Library ini.
resmiAnthropic Academy

Anthropic Academy — Building with the Claude API

Kelas terstruktur gratis dengan sertifikat, mencakup dasar API, tool use, hingga praktik agent — cocok setelah menuntaskan node_02 di Roadmap.
resmiOpenAI

OpenAI Cookbook

Kumpulan notebook resmi berisi pola prompting per model, function calling, RAG, hingga agent — bagus untuk melihat contoh kode nyata, bukan cuma teori.
resmiMCP / Anthropic

Model Context Protocol — Dokumentasi Resmi

Spesifikasi dan panduan protokol terbuka untuk menghubungkan AI ke tools & sumber data eksternal secara standar — rujukan untuk node_02 "AI agent & tool-use".
resmiGoogle Cloud

Google Cloud — Generative AI Learning Path

Jalur belajar generative AI dari Google, termasuk modul gratis "Introduction to Generative AI" dan "Machine Learning Crash Course".
resmiGitHub

openai-cookbook (repositori)

Versi kode sumber dari OpenAI Cookbook — clone lalu jalankan notebook-nya langsung kalau kamu tipe belajar sambil oprek.

Kursus Terstruktur Gratis

Semua bisa diakses tanpa biaya untuk materinya (beberapa sertifikat berbayar), formatnya video + latihan singkat 1–8 jam — pas untuk belajar konsisten tanpa harus cuti kerja.

kursusDeepLearning.AI

DeepLearning.AI Short Courses

Ratusan course singkat gratis (1–2 jam) yang dibuat bareng OpenAI, Anthropic, LangChain, dan Google — mulai dari prompting dasar sampai agentic RAG.
kursusHugging Face

Hugging Face LLM Course

12 chapter gratis dan praktik langsung dengan library Transformers, cocok kalau kamu ingin paham arsitektur di balik LLM, bukan cuma pemakaiannya.
kursusHugging Face

Hugging Face Agents Course

Course gratis dengan sertifikat yang fokus membangun AI agent dari nol — pendamping bagus untuk sub-topik "AI Agent & MCP" di Prompt Library.
kursusMicrosoft

Generative AI for Beginners

21 pelajaran open-source (lisensi MIT) dari prompt engineering, RAG, sampai agent — materinya aktif dirawat komunitas dan gratis penuh.
kursusLangChain

LangChain Academy

Kursus resmi LangChain untuk membangun pipeline RAG dan agent memakai LangGraph — pelengkap teknis untuk sub-topik "RAG" dan "AI Agent".
kursusKaggle

Kaggle Learn

Micro-course gratis plus kompetisi nyata untuk mengasah skill data & machine learning — cocok kalau jalur kamu condong ke Data Analyst/ML.
